Katanya QA ya, mas?

Kaya yang tadi udah disebutin diatas, gua bekerja sebagai karyawan full time Quality Assurance

📋 Hari-hari gua as QA kira-kira ngerjain ginian deh...
REGRESSION TESTING
Ngetest software yang mastiin kalo penambahan fitur atau perbaikan bug ga ngefek ke fitur existing.
✨ Pamali ngerusak yang udah ada soalnya.
SANITY TESTING
Ngetest software yang baru dibuat atau diperbaikin.
✨Kalo masih ada yang salah ya dibalikin aja deh ke dev nya.
USER ACCEPTANCE TESTING
Ngetest dan memverifikasi software nya untuk terakhir kali sebelum dilepas.
✨ Kalo dilepas tanpa kejelasan kasian. Kaya orang linglung nanti.
TEST CASE
Bikin panduan untuk ngetest biar hasil test nanti terukur dan tepat sasaran.
✨ Bayangin kalo ngetest facebook ga pake test case. Bisa-bisa sampe Instagram jadi kena test saking ga teratur alur test nya.
SYSTEM INTEGRATION TESTING
Ngetest beberapa modul yang dikerjakan oleh team apakah bisa terintegrasi dengan benar atau tidak.
✨ Bayangin Tim A ngerjain Pembayaran dan Tim B ngerjain Pengiriman. Tapi karna ga di SIT, bisa-bisa user melakukan pembayaran tapi barang tidak dikirim. SCAM dong jadinya. yaa kasian
BUG REPORTING
Bikin list bug dan how to reproduce bug nya agar tim terkait lebih cepat memahami maksud kita.
✨ Kasian dev nya. Mereka bukan dukun yang harus ngerti semua cara.
